Fluffy Bot (formerly a cat)

Fluffy Bot is a Discord bot that currently supports full functionality with the e621/926 API and features a verification system for new members. It is being developed with a focus on multi-server functionality, and can be configured differently on each server.

What can it do right now?

- Displays a random image based on specified tags, along with post information

- Allows you to configure blacklists/whitelists for both the server and individual channels

- Allows you to manually verify new server members for better control

List of commands:

/e621 [tags] - Random artwork from e621 (18+, the restriction is hardcoded into the bot and cannot be changed)

/2926 [tags] - Random non-18+ artwork from e621

/post_info [post id] - Post information and link to the artwork

/blacklist_add - Add tag(s) to the server's blacklist

/blacklist_remove [tags] - Remove tags from the server's blacklist

/blacklist_show - Show the server's blacklist

/blacklist_clear - Clear the server's blacklist

/verify [user] - Verify a user

/reset_role_permissions - Reset verification role permissions

/set_verification_role [name] - Set the verification role name

/get_verification_role - Show the current role name
