Fluffy Bot (в прошлом был котом)

Fluffy Bot - Discord-бот, на данный момент способный полноценно работать с e621/926 api, а так же имеет систему верификации новых участников. Разрабатывается в том числе с упором на мультисерверность, и на каждом из серверов может быть настроен по-своему.

Что он умеет на данный момент?

- Покажет случайный арт с учетом нужных тегов, с информацией по посту

- Позволяет настроить чёрные/белые листы как для сервера, так и для отдельных каналов

- Позволяет вручную верифицировать новых участников сервера, для лучшего контроля

Список команд:

/e621 [теги] - Случайный арт с е621 (18+, ограничение вписано в бота и не меняется)

/2926 [теги] - Случайный не 18+ арт с е621

/post_info [id поста] - Информация о посте и ссылка на арт

/blacklist_add - Добавить тег (и) в blacklist сервера

/blacklist_remove [теги] - Удалить теги из blacklist сервера

/blacklist_show - Показать blacklist сервера

/blacklist_clear - Очистить blacklist сервера

/verify [пользователь] - Верифицировать пользователя

/reset_role_permissions - Сбросить права роли верификации

/set_verification_role - [название] Установить название роли для верификации

/get_verification_role - Показать текущее название роли
